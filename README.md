HTML5のcanvasとfile APIを使ったグラフ描画ツールです。

##特長
* ファイルをdrag&dropするだけでOK
* データは単純なCSVでOK、色やラベルをコントロールする場合はJSONで
* 複数のファイルを１画面に描画可、１ファイルに複数のデータも記述可
